Why do we need JOIN 

Databases store data in separate tables to avoid repetition. 
JOIN connects those tables when you need combined information. 

Example 

Employee table ‚Üí employee details 

Department table ‚Üí department details 

JOIN helps you see which employee works in which department. 

 

Basic JOIN syntax 

SELECT columns 
FROM table1 
JOIN table2 
ON table1.column = table2.column;
